# QR-Based Restaurant Ordering System

A full-stack QR-based restaurant ordering web application where customers can place orders without logging in, and restaurant staff can manage menu items and incoming orders through a secure dashboard. This project simulates a real-world caf√©/restaurant ordering flow using modern web technologies.

## Features

- Place orders without requiring customer login
- QR-based menu access for tables
- Admin dashboard to manage menu items and view incoming orders

## Tech stack

- Frontend: React (Vite)
- Backend: Python
